Repo2Markdownfile

Paste a public GitHub repo. Download one structured markdown file.

This GitHub Pages app reads a public GitHub repository in your browser and turns it into one markdown file that is easy to upload into NotebookLM, Gemini, ChatGPT, Claude, or another long-context LLM.

Preparing...

What it is for

Use the markdown file for a deep dive on a repository, quick familiarization before an interview, study plans, onboarding notes, or podcast prep.

Great with

NotebookLM, Gemini, ChatGPT projects, Claude, and other long-context tools that work better with one clean document than scattered repo pages.

Best for

Public repositories of any size, with the understanding that large repositories may take longer because everything runs in the browser.

Examples:

  • https://github.com/owner/repo
  • https://github.com/owner/repo/tree/main
  • https://github.com/owner/repo/tree/main/src
