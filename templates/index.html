Repo2Markdown

Turn any public GitHub repo into one LLM-ready markdown file.

Paste a public GitHub repository, branch, or folder URL. The app downloads the archive, turns the repository into one structured markdown document, and gives you a clean file you can upload to NotebookLM, Gemini, or another long-context LLM for a fast deep dive.
